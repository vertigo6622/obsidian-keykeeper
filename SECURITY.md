## security features

this application contains various security measures to keep data protected and safe. however, nothing is impenetrable. here are some of these features:

* extensive input validation
* rate limiting and regex throughout the application
* automatic account suspensions for license violations
* tor-over-clearnet architecture to hide the location and ip of the backend
* keeps track of account age and general standing
* uses an obscure but powerful and lightweight cipher SPECK for verification and encryption
* satoshi and piconero based calculations for crypto payments to help prevent rounding abuse
* hardened content security policy and hsts
* origin and file access whitelisting
* protections against race conditions
